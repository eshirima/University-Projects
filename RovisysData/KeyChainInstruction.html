Keychain Class Wrapper Instructions

How to Use

  • Perform a pod install first
  • Realm Deletion

    • Uncomment the line User().deleteEverythingFromRealm() inside the didFinishLaunchingWithOptions(). This can be found inside the AppDelegate.swift file

  • Launch the app

    • Upon a successful launch, you should see prompts on the console of realm objects

  • Stop the app

    • Comment out the method in step II

  • Add Users to Realm

    • Navigate to the ViewController.swift file. Locate oldLoginStuff() and comment out everything else except for the user creation portions. Then uncomment the function call inside the viewDidLoad()

  • Repeat step III then quit simulator
  • Add Projects to Realm

    • Go back to oldLoginStuff() and comment out the user creation portions and uncomment everything else. Run the app then quit simulator upon a successful app launch. Then comment out the oldLoginStuff() inside the viewDidLoad()

  • Voila, everything is encrypted!
